## 🎨 Design System Components

### Typography Scale:
```
Display:    36sp Bold     (Headlines)
Headline:   28sp Bold     (Screen titles)
Title:      20sp Medium   (Card titles)
Body:       16sp Regular  (Main content)
Caption:    14sp Regular  (Secondary text)
Small:      12sp Light    (Hints/metadata)
```

### Spacing System:
```
XS:  4dp   (Tight spacing)
SM:  8dp   (Compact elements)
MD:  16dp  (Standard padding)
LG:  24dp  (Section gaps)
XL:  32dp  (Large margins)
XXL: 48dp  (Hero spacing)
```

### Corner Radius:
```
Buttons:     12dp
Cards:       16dp
Inputs:      12dp
Chips:       16dp (full height)
Avatars:     50% (circular)
```

### Elevation Levels:
```
Level 0: Flat (backgrounds)
Level 1: 2dp (cards)
Level 2: 4dp (raised elements)
Level 3: 8dp (FAB, modals)
Level 4: 16dp (dialogs)
```
